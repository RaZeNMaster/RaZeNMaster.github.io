- **Vanilla HTML / CSS / JS** — everything lives in a single `index.html`.

- Tech icons via [devicon](https://devicon.dev/).
- Minimalist, scrollable layout with numbered sections and code-styled accents.

## Run locally

No build step — just open the file or serve the folder:

```bash
# option 1: open directly
start index.html        # Windows
# option 2: tiny local server
python -m http.server 8000
# → http://localhost:8000
```

## Deployment

Hosted on **GitHub Pages** from the `main` branch. Because the repository is
named `RaZeNMaster.github.io`, it's served directly at the user root URL —
every push to `main` updates the live site automatically.
